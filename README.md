# Study Tracker Application
---------------------------------------------------------------------------------------------------------

A console-based Java application to record and manage daily study activities.
Users can add study logs, view reports, generate summaries, and export data to a CSV file.

* Application Menu :
  
1. Insert new study log
2. View all study logs
3. Summary of study log by date
4. Summary of study log by subject
5. Export study log to CSV file
6. Exit

* CSV Export

File name: StudyTracker_Report.csv
- Generated in the same project directory

# Concept Used

  - packages 
  - classes,objects
  - collections
  - file handling
  - user input handling.

# How to Compile and Run

  * Compile :
    javac Study_Tracker/*.java
    
  * Run :
    java Study_Tracker.StudyTrackerStarter
